## PROJECT DESCRIPTION

This project is a Sports Academy Management System written in C.  
It manages student enrollment across multiple sports with limited seats per sport and stores records using file operations.

Supported Sports:
- Badminton
- Cricket
- Basketball
- Hockey
- Golf
- Swimming
- Table Tennis
- Football

Each student record stores:
- Name
- SAP ID
- Roll ID
- Address
- Height
- Weight
- Enrolled Sport
